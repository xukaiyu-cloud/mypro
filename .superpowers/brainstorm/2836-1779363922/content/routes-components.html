路由设计与组件树

Vue Router 路由划分 + 核心组件嵌套关系

📋 路由表设计
/ → MainLayout（主布局）
/login → LoginPage（登录）
/register → RegisterPage（注册）
/import → ImportPage（导入歌单）
/settings → SettingsPage（设置）
/about → AboutPage（关于）

主布局 MainLayout 内部子路由：
  / → 默认：播放队列视图
  /playlist/:id → 歌单详情
  /sync → 同步视图
  /history → 播放历史
  /search → 搜索结果
  /favorites → 收藏列表

路由设计要点

路由守卫
登录/注册页面对外开放；
主界面、导入、同步等页面需要 登录状态校验，未登录自动跳转 /login
布局复用
MainLayout 作为壳布局，PlayerBar 固定在底部不随路由切换销毁，保证播放不中断
